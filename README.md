# Technical Test Beon Intermedia

## HOW TO RUN BACKEND ##
1. cd backend
2. setup database on file ".env"
3. php artisan migrate
4. php artisan db:seed
5. php artisan serve

**ARCHITECTURAL PATTERN**
1. Service pattern
2. Slim controller, Fat service

4. pnpm dev
